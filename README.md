### ✨사이트 소개
펫 간식 브랜드 '힐링펫'의 상품과 판매방식 등을 참고하여 만든 사이트입니다.

Cherish는 '소중히 여기다'라는 뜻을 가지고 있습니다.

애완동물을 소중히 여기자는 뜻으로 브랜드명을 만들게 되었습니다.
